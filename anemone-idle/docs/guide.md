# 말미잘 방치형: 뿅 회사 게임 가이드

`말미잘 방치형: 뿅 회사`는 사장 말미잘을 찰싹거리며 꿈틀밥을 벌고, 수상한 바다 직원들을 고용해서 자동 수익을 늘리는 병맛 방치형 게임입니다.

## 실행 방법

```bash
cd /Users/a1/Desktop/manus/anemone-idle
npm run dev
```

브라우저에 뜨는 로컬 주소로 접속하면 됩니다. 기본 포트가 이미 사용 중이면 Vite가 자동으로 다음 포트를 잡습니다.

## 기본 목표

핵심 목표는 간단합니다.

1. `사장 말미잘 찰싹!` 버튼을 눌러 `꿈틀밥`을 번다.
2. 꿈틀밥으로 생산자를 고용한다.
3. 생산자가 자동으로 자원을 벌어오게 만든다.
4. 장 목표 보상을 받고, 전리품 조개를 열고, 허세 진주 업그레이드를 산다.
5. 더 많은 자동 수익을 만든다.

## 자원

| 자원 | 영어명 | 용도 |
|---|---|---|
| 꿈틀밥 | Wiggle Food | 기본 돈. 생산자 고용에 주로 사용합니다. |
| 허세 진주 | Drama Pearls | 업그레이드와 고급 생산에 사용합니다. |
| 뿅 파워 | Bloop Power | 일부 고급 업그레이드에 사용합니다. |

## 수동 수익

`노동 산호초` 탭의 큰 버튼 `사장 말미잘 찰싹!`을 누르면 즉시 꿈틀밥을 얻습니다.

현재 수동 보상은 최소 꿈틀밥 `+3`에서 시작하며, 현재 꿈틀밥 초당 생산량 약 10초치가 더해져 함께 커집니다. 생산자를 늘릴수록 버튼 보상도 같이 강해집니다.

## 방치 수익

이 게임은 시작하자마자 기본 `인턴 촉수` 1개가 있어 바로 방치 수익이 생깁니다.

추가 생산자를 고용하면 자동 수익이 더 커지고, 그 생산량을 기준으로 `사장 말미잘 찰싹!` 보상도 커집니다.

게임을 꺼두거나 다른 일을 하다가 돌아오면 오프라인 진행 보상이 적용됩니다. 오프라인 보상은 최대 4시간까지만 쌓입니다.

## 생산자

| 생산자 | 내부 역할 | 설명 |
|---|---|---|
| 인턴 촉수 | 꿈틀밥 생산 | 가장 기본 생산자입니다. 초반 성장의 핵심입니다. |
| 새우 중간관리자 | 꿈틀밥 생산 | 비싸지만 인턴 촉수보다 훨씬 많이 법니다. |
| 조개 인플루언서 농장 | 허세 진주 생산 | 허세 진주를 천천히 벌어옵니다. |
| 수상한 달빛 호스 | 뿅 파워 생산 | 고급 자원인 뿅 파워를 만듭니다. |

생산자는 살 때마다 다음 구매 가격이 올라갑니다. 방치형 게임 특성상 “많이 사서 더 많이 벌고, 더 비싼 걸 산다”가 기본 루프입니다.

## 1장: 업무 산호초 창업

상단의 가로 카드들은 이제 `장 목표`입니다. Chapter 1은 `1장: 업무 산호초 창업`이고, 단순히 자원을 쌓는 대신 다음 해금과 현재 병목을 계속 보여줍니다.

`노동 산호초` 탭에는 장 업무판이 표시됩니다.

- 현재 장 진행률
- 다음 해금 목표
- 현재 병목 자원
- 현재 생산량
- 대게/고래 같은 자동 고용 생산자의 진행률

예시:

- 인턴 촉수 5개 보유
- 새우 중간관리자 3마리 보유
- 전리품 조개 보상 1회 받기
- 허세 진주 업그레이드 구매
- 대게 지사장 1마리 보유

목표 조건을 달성하면 버튼이 보상 받기 상태로 바뀝니다. 보상은 초반 성장에 중요하니 자주 확인하세요. 1장을 모두 끝내면 빈 목록 대신 `1장 업무 산호초 완료` 카드가 나오고, 다음 커뮤니티 원정이 준비 중이라고 안내합니다.

## 전리품 조개

`전리품 조개` 탭에서는 시간제 캡슐을 시작할 수 있습니다.

1. 조개 흔들기를 시작한다.
2. 타이머가 끝날 때까지 기다린다.
3. 조개를 열어 자원을 받는다.

전리품 조개 보상은 현재 생산자 상태에 따라 달라집니다. 생산자가 많을수록 보상이 좋아집니다.

## 허세 진주 업그레이드

`허세 진주` 탭에서 업그레이드를 살 수 있습니다.

| 업그레이드 | 효과 |
|---|---|
| 임원용 국수 팔 | 모든 생산자 생산량 25% 증가 |
| 프리미엄 무지개 핑계 | 모든 생산자 생산량 35% 증가 |
| 조개 전자레인지 모드 | 전리품 조개 숙성 시간 35% 단축 |

업그레이드는 한 번 사면 계속 적용됩니다.

## 언어 변경

`언어 도깨비` 탭에서 영어와 한국어를 전환할 수 있습니다.

선택한 언어는 브라우저 저장소에 저장되므로 새로고침해도 유지됩니다.

## 초반 추천 루트

1. 기본 `인턴 촉수`가 벌어오는 자동 수익을 확인합니다.
2. `사장 말미잘 찰싹!`과 자동 수익을 같이 써서 인턴 촉수를 5개까지 늘립니다.
3. 생산자가 늘어나면 찰싹 보상도 같이 커지는지 확인합니다.
4. 첫 미션 보상을 받습니다.
5. `새우 중간관리자`를 목표로 꿈틀밥을 모읍니다.
6. 전리품 조개를 계속 돌려 보너스 자원을 챙깁니다.
7. 허세 진주가 생기면 업그레이드를 삽니다.

## 저장

게임은 `localStorage`에 자동 저장됩니다.

저장되는 내용:

- 보유 자원
- 생산자 수
- 구매한 업그레이드
- 완료한 미션과 장 목표 진행 상태
- 진행 중인 전리품 조개
- 선택한 언어
- 마지막 저장 시각

브라우저 저장소를 지우면 진행 상황도 초기화됩니다.

## 개발/검증 명령어

```bash
npm test
npm run build
node scripts/validate-progression-content.mjs
node scripts/balance-report.mjs --scenario chapter1-active
```

현재 핵심 로직은 Vitest와 검증 스크립트로 확인합니다. 테스트는 경제, 캡슐, 기존 미션 호환, 저장 마이그레이션, 진행 그래프, 밸런스 타깃, 번역 커버리지를 확인합니다. `validate-progression-content`는 퀘스트 그래프와 참조 오류를 검사하고, `balance-report`는 0-15분 Chapter 1 루프가 목표 수치에 맞는지 `.omo/evidence` 아래 JSON으로 남깁니다.
