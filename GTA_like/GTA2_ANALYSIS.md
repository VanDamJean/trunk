# GTA 2 게임 요소 분석 (Reference Document)

> 이 문서는 GTA 2 (1999, DMA Design)의 핵심 요소를 분석하고,
> 우리 프로젝트에서 구현할 범위를 정리하기 위한 레퍼런스 자료입니다.

---

## 1. 시점 및 카메라 🎥

| 항목 | GTA 2 원작 | 비고 |
|------|-----------|------|
| 기본 시점 | 탑다운(Top-down) | 완전히 위에서 내려다보는 새의 눈 시점 |
| 카메라 줌 | **동적 줌** - 도보일 때 가깝게, 차량 탑승 시 멀리 | 속도에 따라 자동 조절 |
| 3D 엔진 | 2.5D (3D 엔진으로 구동, 탑다운 렌더링) | 건물에 높이감, 램프에서 점프 가능 |
| 조명 | 실시간 다이나믹 라이팅 | 가로등, 헤드라이트, 폭발 이펙트 |

## 2. 캐릭터 조작 🕹️

### 도보 (On Foot)
- **이동**: 전진, 후진, 좌우 회전 (탱크 컨트롤)
- **달리기**: 별도 키로 달리기 가능
- **점프**: 장애물 뛰어넘기, 높은 곳에서 떨어지기
- **무기 사용**: 바라보는 방향으로 발사
- **상호작용**: 차량 탑승, 아이템 픽업

### 차량 (In Vehicle)
- **가속/브레이크**: 전진, 후진
- **좌우 조향**: 핸들 꺾기
- **하차**: 차에서 내리기
- **차량 무기**: 기관총, 지뢰 (장착 시)

## 4. 차량 시스템 🚗

### 차량 종류 (총 69종)
| 카테고리 | 예시 | 특징 |
|----------|------|------|
| 스포츠카/머슬카 | A-Type, Arachnid | 고속, 드리프트 |
| 일반 승용차 | 다수 | 시내에 가장 흔함 |
| 경찰차 | Police Car | 경찰 추격 시 등장 |
| SWAT 밴 | SWAT Van | 수배 4단계~ |
| FBI 차량 | Special Agent Car | 수배 5단계~ |
| 탱크 | Tank | 수배 6단계(최고), 플레이어도 탈취 가능 |
| 버스/택시 | Bus, Taxi | 대중교통 |
| 리무진 | Limousine | 갱 보스 이용 |
| 트럭/밴 | Box Truck, Garbage Truck | 느리지만 내구성 높음 |
| 갱 전용 차량 | Z-Type (Zaibatsu) 등 | 각 갱마다 고유 차량 |
| 특수 차량 | 소방차, 구급차, Ice Cream Van | 특수 미션 관련 |

### 차량 물리
- **가속/최고속**: 차량 종류별 상이
- **핸들링**: 스포츠카는 예민, 트럭은 둔감
- **내구도**: 충돌/총격 시 데미지 → 연기 → 폭발
- **탈취**: 도로 위 모든 차량을 뺏을 수 있음 (시민/갱/경찰 차 모두)

## 5. 수배 시스템 (Wanted Level) ⭐

| 레벨 | 대응 세력 | 설명 |
|------|----------|------|
| ☆ (0) | 없음 | 평화 상태 |
| ★ (1) | 경찰 | 따라오며 정지 시도 |
| ★★ (2) | 경찰 (공격적) | 총격 시작 |
| ★★★ (3) | 경찰 (다수) | 적극적 사살 시도 |
| ★★★★ (4) | SWAT | SWAT 밴과 중무장 팀 투입 |
| ★★★★★ (5) | FBI | 특수요원 차량, 더 빠르고 강력 |
| ★★★★★★ (6) | 군대 (National Guard) | 탱크, 무장 지프 투입 - 최고 난이도 |

### 수배 레벨 상승 조건
- 시민 공격/살해
- 차량 탈취 (경찰이 목격 시)
- 경찰/SWAT/FBI 공격
- 차량 파괴
- 폭발 유발

### 수배 레벨 감소/해제 방법
- **도주**: 경찰 시야에서 벗어나 일정 시간 유지
- **리스프레이 가라지 (Max Paynt)**: $5,000 - 차량 수리 + 수배 해제
- **체포 (Busted)**: 수배 해제되지만 무기 전부 몰수
- **사망 (Wasted)**: 수배 해제, 무기 몰수, 돈 감소

## 6. 갱단 시스템 (Gang Respect) 🤝

### 3개 구역, 각각 3개 갱단
| 구역 | 갱단 1 | 갱단 2 | 갱단 3 |
|------|--------|--------|--------|
| Downtown (상업) | Zaibatsu | Loonies | Yakuza |
| Residential (주거) | Zaibatsu | Rednecks | SRS Scientists |
| Industrial (공업) | Zaibatsu | Russian Mafia | Hare Krishna |

> **Zaibatsu**는 모든 구역에 존재하는 유일한 갱단

### 호감도 (Respect) 메커니즘
- 각 갱단에 대해 **-3 ~ +3** 단계의 호감도 존재
- **호감도 상승**: 해당 갱의 미션 완료, 라이벌 갱단원 사살
- **호감도 하락**: 해당 갱단원 공격, 라이벌 갱 미션 수행
- **호감도 높을 때**: 더 좋은 미션 해금, 영역 내 안전
- **호감도 낮을 때**: 영역 진입 시 즉시 공격당함

## 7. 미션 시스템 📞

### 미션 수주 방법: 색깔별 공중전화
| 색상 | 난이도 | 필요 호감도 | 보상 |
|------|--------|------------|------|
| 🟢 초록 | 쉬움 | 없음 | 낮음 |
| 🟡 노랑 | 보통 | 중간 | 중간 |
| 🔴 빨강 | 어려움 | 최대 | 높음 |

### 미션 유형
- **운송**: 특정 차량/물건을 목적지까지 배달
- **파괴**: 차량/건물 파괴
- **암살**: 특정 타겟 제거
- **레이싱**: 차량 경주
- **보호**: NPC/차량 호위
- **복합**: 여러 단계가 결합된 체인 미션

### 진행 시스템
- 미션 완료 → 돈 획득 → **목표 금액 달성 시 다음 구역(District) 해금**
- 필수 미션은 없음. 자유롭게 돈을 모으면 됨

| 요소 | 위치 | 설명 |
|------|------|------|

| 방어구 (🛡️) | 우상단 | 아머 수치 |
| 현재 무기 / 탄약 | 좌상단 | 선택된 무기와 잔탄 |

| 갱 호감도 | 좌하단 | 현재 구역 갱단 3개의 호감도 바 |
| 방향 나침반 | 하단 | 미니맵 없음! 목적지 방향만 화살표로 표시 |

## 11. 라디오 / 사운드 🎵

- **라디오 스테이션**: 차량 탑승 시 자동 재생
- 각 갱단 영역에서 해당 갱의 라디오가 수신됨
- 풍자적인 광고, DJ 멘트 포함
- 차량 엔진음, 총격음, 폭발음, 시민 비명 등 효과음

## 12. 기타 메커니즘

### 생존 / 사망
- **사망 (Wasted)**: 체력 0 → 병원에서 리스폰, 무기 몰수, 돈 감소
- **체포 (Busted)**: 경찰에 잡힘 → 경찰서에서 리스폰, 무기 몰수

### 세이브 시스템
- **교회**에서만 저장 가능, **$50,000** 필요
- 자동 저장 없음 → 긴장감 유지

### 멀티플레이어 (PC)
- 최대 6인 온라인 대전
- 전용 맵에서 데스매치

## 📝 우리 프로젝트에서 구현 가능 범위 (잠정)

이 분석을 바탕으로 다음 문서(`GAME_DESIGN.md`)에서
구현 우선순위와 세부 설계를 정리할 예정.

- [ ] 탑다운 시점 + 동적 카메라 줌

- [ ] 차량 탑승/하차/운전 물리
- [ ] 기본 무기 2~3종 (주먹, 권총, 기관총)

- [ ] 시민 NPC (도보)
