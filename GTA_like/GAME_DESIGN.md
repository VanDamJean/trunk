## 1. 게임 컨셉

### 1.1 핵심 루프
```
도시 탐험 → 범죄/미션 수행 → 돈 획득 → 무기/차량 업그레이드
     ↑                                        ↓
     ← ← ← ← 새로운 구역 해금 ← ← ← ← ← ← ←
```

### 1.2 차별점 (GTA 2 원작 vs 우리 게임)
| 요소 | GTA 2 원작 | 우리 게임 |
|------|-----------|----------|
| 사격 | 수동 조준/발사 | **뱀서 모드**: 자동 조준 + 자동 발사 |
| 무기 성장 | 픽업으로 교체 | 경험치 → 레벨업 → 무기 진화 |
| 난이도 | 고정 | 시간 경과에 따라 적 물량 증가 (뱀서식) |
| 조작 | 키보드/패드 | **모바일 최적화** (가상 조이스틱) |
| 세이브 | 교회 ($50,000) | 자동 저장 |

### 1.3 게임 모드
- **스토리 모드**: GTA 2식 미션 진행, 구역 해금
- **뱀서 모드**: 시간 제한 서바이벌, 자동 사격, 웨이브 생존
- **자유 모드**: 미션 없이 샌드박스 플레이

### 2.2 게임 루프
```python
# 목표 FPS: 60
while running:
    dt = clock.tick(60) / 1000.0  # 델타 타임
    
    handle_input(dt)      # 입력 처리
    update_entities(dt)   # 엔티티 업데이트
    update_systems(dt)    # 시스템 업데이트 (수배, 갱단, 미션)
    check_collisions()    # 충돌 감지
    render(dt)            # 렌더링
    update_ui()           # HUD 업데이트
```

## 4. 차량 시스템

### 4.1 차량 카테고리 (구현 대상)
| 티어 | 이름 | 최고속 | 가속 | 핸들링 | 내구도 |
|------|------|--------|------|--------|--------|
| S | 스포츠카 | 10 | 0.3 | 4.0 | 60 |
| A | 경찰차 | 8 | 0.25 | 3.5 | 80 |
| A | 갱 전용차 | 9 | 0.25 | 3.0 | 70 |
| B | 일반 승용차 | 6 | 0.2 | 3.0 | 80 |
| B | 택시 | 6 | 0.2 | 3.0 | 80 |
| C | 트럭/밴 | 4 | 0.1 | 2.0 | 150 |
| C | 버스 | 3 | 0.08 | 1.5 | 200 |
| S+ | SWAT 밴 | 7 | 0.2 | 2.5 | 200 |
| S++ | 탱크 | 3 | 0.05 | 1.0 | 500 |

### 4.2 차량 물리
```
가속: speed += acceleration * dt  (max_speed 제한)
감속: speed -= friction * dt      (0 이하 시 정지)
조향: angle += turn_speed * dt    (speed > 0.1 일때만)
충돌: hp -= impact_force          (연기 → 화재 → 폭발)
```

### 4.3 차량 상태
```
[정상] → hp < 50% → [연기] → hp < 25% → [화재] → hp = 0 → [폭발]
                                          (5초 후 자동 폭발)
```

### 4.4 차량 서비스 (가라지)
| 서비스 | 비용 | 효과 |
|--------|------|------|
| 리스프레이 | $5,000 | 수리 + 수배 해제 |
| 폭탄 장착 | $10,000 | 하차 후 원격 폭파 |
| 기관총 장착 | $25,000 | 전방 자동 사격 |
| 지뢰 장착 | $50,000 | 후방 투하 (10개) |

## 6. 수배 시스템

### 6.1 단계별 대응
```
★☆☆☆☆☆  경찰 1~2명, 걸어서 접근
★★☆☆☆☆  경찰차 1대 추격
★★★☆☆☆  경찰차 2~3대, 총격
★★★★☆☆  SWAT 밴 등장, 중무장
★★★★★☆  FBI 차량, 더 빠르고 정확
★★★★★★  탱크 + 무장 지프 (지옥)
```

### 6.2 수배 증가 트리거
| 행위 | 증가량 | 조건 |
|------|--------|------|
| 시민 폭행 | +0.3 | 경찰 시야 내 |
| 시민 사살 | +1.0 | - |
| 차량 탈취 | +0.5 | 경찰 시야 내 |
| 경찰 공격 | +1.0 | - |
| 차량 폭발 | +0.5 | - |
| 경찰 사살 | +2.0 | - |

### 6.3 수배 감소
| 방법 | 효과 |
|------|------|
| 시간 경과 (은신) | -0.1/초 (경찰 시야 밖) |
| 리스프레이 가라지 | 즉시 0으로 |
| 체포 (Busted) | 즉시 0, 무기 몰수 |

## 7. NPC & AI

### 7.1 시민 AI
```
[배회] ──(플레이어 총격 감지)──→ [도주]
  ↑                                ↓
  └────(안전 거리 도달)────────────┘
```
- 도로/인도를 따라 무작위 이동
- 총격/폭발 감지 시 반대 방향으로 도주
- 차량 운전 시민: 도로 위를 주행, 신호 대기

### 7.2 경찰 AI
```
[순찰] ──(수배 발생)──→ [추격] ──(접근)──→ [공격/체포]
                          ↓
                    (시야 상실) → [수색] → [순찰 복귀]
```
- 수배 레벨에 따라 공격 패턴 변화
- 레벨 1~2: 접근 후 체포 시도
- 레벨 3+: 즉시 사격

### 7.3 갱단원 AI
```
[영역 순찰] ──(적대 플레이어)──→ [공격]
     ↓
(우호 플레이어) → [무시/환영]
```

## 12. 뱀서 모드

### 12.1 컨셉
GTA 2의 세계관 + 뱀파이어 서바이버즈의 자동 전투

### 12.2 자동 사격 로직
```python
# 매 fire_cooldown 프레임마다 실행
closest = find_nearest_enemy(range=400)
if closest:
    angle = atan2(closest.y - player.y, closest.x - player.x)
    for i in range(bullet_count):
        offset = (i - (bullet_count-1)/2) * spread_angle
        spawn_bullet(player.pos, angle + offset)
```

### 12.3 레벨업 보상 (랜덤 3개 중 택 1)
| 업그레이드 | 효과 | 최대 |
|-----------|------|------|
| 연사 속도 UP | 쿨다운 -5프레임 | 10프레임 |
| 멀티샷 | 총알 +1발 | 8발 |
| 관통 | 총알이 적 관통 | 3회 |
| 데미지 UP | 데미지 +20% | 5회 |
| 사거리 UP | 사거리 +50px | 5회 |
| 이동속도 UP | 이동 +0.5 | 3회 |
| 체력 회복 | HP +20 회복 | 무제한 |
