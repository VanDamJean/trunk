# Qwen Tetris

pygame으로 만든 네온 스타일 테트리스 게임입니다.

## 설치

```bash
pip install pygame
```

## 실행

```bash
cd qwen_tetris
python3 tetris.py
```

## 조작법

| 키 | 동작 |
|---|---|
| ←/→ | 좌우 이동 |
| ↓ | 소프트 드롭 (조금 빨리 내리기) |
| ↑ | 회전 |
| Space | 하드 드롭 (즉시 낙하) |
| C | 홀드 (현재 블록 저장) |
| ESC | 일시정지 |

## 기능

- **고스트 피스**: 블록이 떨어질 위치를 미리 보여줍니다
- **홀드 시스템**: 'C' 키로 현재 블록을 저장하고 나중에 사용
- **레벨 시스템**: 10줄 삭제마다 레벨 업, 속도가 점진적으로 빨라짐
- **점수 시스템**: 동시 삭제 줄 수에 따라 차등 점수 (1줄=100, 2줄=300, 3줄=500, 4줄=800) x 레벨
- **일시정지**: ESC 키로 게임 일시정지
- **재시작**: 게임 오버 후 키 입력으로 바로 재시작 가능
