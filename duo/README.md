# DUO (듀오) - 단어 반복학습 앱

하루 15분, 과학적 간격 반복 알고리즘으로 단어를 자동으로 외워주는 모바일 우선 웹 애플리케이션입니다.

## 기능
- **간격 반복 스케줄링**: FSRS 알고리즘을 사용해 개인별 최적의 복습 타이밍을 계산합니다.
- **다양한 퀴즈 타입**: 플래시카드, 객관식, 빈칸 채우기, 매칭 등 학습 수준에 맞춰 난이도가 자동 조절됩니다.
- **게이미피케이션**: 스트릭(연속 학습일), XP, 레벨 시스템으로 동기를 부여합니다.
- **오프라인 지원**: PWA 및 LocalStorage 기반으로 오프라인에서도 작동합니다. (단, TTS는 브라우저 지원 필요)
- **오디오 피드백**: 정답/오답 효과음 및 Web Speech API 기반 단어 읽어주기(TTS) 기능을 제공합니다.

## 기술 스택
- **프론트엔드**: HTML, Vanilla JavaScript, CSS (No Framework)
- **빌드 도구**: Vite
- **핵심 라이브러리**: `ts-fsrs` (간격 반복 스케줄링)
- **데이터베이스**: 브라우저 LocalStorage

## 실행 방법

### 요구사항
- Node.js (v18 이상 권장)

### 설치 및 실행
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 프로덕션 빌드
```bash
# 빌드
npm run build

# 로컬에서 빌드된 결과물 확인
npm run preview
```

## 디렉토리 구조
- `src/data/`: 단어 데이터
- `src/lib/`: FSRS 스케줄러, 로컬 스토리지, 게이미피케이션 등 핵심 로직
- `src/screens/`: 각 화면 뷰 (홈, 학습, 리그, 단어장 등)
- `src/components/`: UI 컴포넌트
- `src/style.css`: 디자인 시스템

## 주의사항
- 본 앱은 브라우저의 `LocalStorage`에 모든 데이터를 저장합니다. 브라우저 데이터를 삭제하면 학습 기록이 초기화될 수 있습니다.
