# 피그마 Word Game (Community) - 5개 화면 분석

## 화면 1: 메인 게임 화면 (Main Game Screen)
- **상단 헤더**: 별(점수), 레벨, 보석 수
- **중앙**: 리들 질문 텍스트
- **답 슬롯**: 5개 빈 박스
- **버튼**: "Жардам" (도움말) 버튼
- **문자 그리드**: 4x2 레이아웃 (8개 문자)
- **배경**: 밝은 하늘색, 구름, 태양, 산, 나무, 유르트

## 화면 2: 도움말 모달 (Help Modal)
- **제목**: "Жардам" (도움말)
- **내용**: 
  - "Асыл таш табышмалары" (귀중한 리들들)
  - 설명 텍스트
- **버튼**: 
  - "Таңга ачуу" (문자 열기) - 파란색
  - "Таңга өчүрүү" (문자 제거) - 파란색
- **닫기**: X 버튼

## 화면 3: 게임 진행 중 (In-Game Progress)
- **상단 헤더**: 별(점수), 레벨, 보석 수
- **중앙**: 리들 질문
- **답 슬롯**: 일부 채워진 상태 (예: 3개 문자)
- **버튼**: "Жардам" + "Өчүрүү" (제거) 버튼
- **문자 그리드**: 일부 문자 비활성화 (회색)

## 화면 4: 보석 모달 (Gem Modal)
- **제목**: "Асыл таш" (보석)
- **내용**: 보석 획득 정보
- **버튼**: 
  - "Асыл змес" (보석 받기) - 파란색
  - "Мекул" (시청) - 파란색
- **닫기**: X 버튼

## 화면 5: 성공 모달 (Success Modal)
- **제목**: "БУРКУТ" (독수리 - 축하)
- **내용**: 
  - 별 3개 표시 (⭐⭐⭐)
  - "Туура жооп!" (정답!)
  - "+2" 보석 표시
- **버튼**: "Улантуу" (계속) - 파란색
- **배경**: 흰색 박스, 파란색 테두리

## 주요 UI 요소 정리

### 색상
- 배경: 밝은 하늘색 (#AEE2FF)
- 헤더: 파란색 그래디언트 (#2196F3 → #1976D2)
- 버튼: 파란색 (#2196F3)
- 문자 버튼: 밝은 파란색 (#93C5FD)
- 비활성화: 회색 (#9CA3AF)

### 모달 구조
- 반투명 검은색 오버레이
- 흰색 배경 박스
- 둥근 모서리
- 그림자 효과

### 버튼 유형
1. **주요 버튼**: 파란색, 둥근 모양
2. **보조 버튼**: 파란색, 작은 크기
3. **닫기 버튼**: X 아이콘
