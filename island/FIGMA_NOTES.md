## 게임 로직 — 매치-3 (Match-3)

### 개요
캔디크러시 계열 **스왑 매치-3** 게임.  
인접한 젬 두 개를 교환해 가로/세로 3개 이상 같은 색을 맞추면 제거된다.

### 그리드
- 크기: **6 × 6** (36칸)
- 젬 색상 4종: 파랑(b) / 핑크(p) / 노랑(y) / 초록(g)
- 초기화 시 3-in-a-row가 없도록 배치

### 입력 방식 — 드래그 스왑

```
pointerdown (젬 터치/클릭)
  → 젬 선택 표시 (흰 glow)
pointermove
  → 젬이 손가락/커서를 실시간으로 따라 이동 (translate)
pointerup
  → 드래그 방향(상/하/좌/우) 판별
  → 인접 칸으로 스왑 시도
  → 드래그 거리 < 18px이면 무시 (오터치 방지)
```

### 턴 구조

```
드래그 → 스왑 시도
   ├─ 매치 없음    → 350ms 후 되돌림 (무브 소진 없음)
   └─ 매치 있음    → 제거 → 낙하(Gravity) → 연쇄 체크
                     Moves -1
```

### 매치 판정
- 가로 3개 이상 동일 색 → 전부 매치
- 세로 3개 이상 동일 색 → 전부 매치
- 교차점은 중복 포함 (T/L형 동시 처리)

### 낙하 (Gravity)
매치 제거 후 빈 칸 위의 젬들이 아래로 떨어지고, 상단 빈 칸에는 랜덤 젬 채움.

### 연쇄 (Cascade)
낙하 완료 후 다시 매치 탐색 → 있으면 자동 제거 반복 (무브 추가 소진 없음).

### 목표 (Target)
- 파란 젬(b)을 매치로 제거할 때마다 Target 카운터 -1
- Target = 0 → 레벨 클리어 → You Win 팝업

### 무브 (Moves)
- 스왑 성공(매치 발생) 시에만 Moves -1
- Moves = 0 도달 시 → You Win 팝업 (점수 기준 결과)

### 점수 (Score)
| 상황 | 점수 |
|---|---|
| 기본 1개 제거 | +100 |
| 매치 크기 보너스 | +10 × 매치 수 |
