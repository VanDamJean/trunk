# Art Direction Overhaul Brief: "Cartoon Hybrid-Casual RPG"

본 문서는 Yacoo RPG Android 버전의 비주얼 퀄리티를 글로벌 메가 히트 캐주얼 게임(예: *탕탕특공대*, *궁수의 전설*) 수준으로 끌어올리기 위한 아트 가이드라인입니다. 

---

## 1. 아트풍 전체 정의 (Art Style & Tone)

전체적인 비주얼 컨셉은 **"두꺼운 외곽선의 카툰/코믹스 스타일(Bold Outline Cartoon)"**과 **"고대비 하이브리드 캐주얼 UI"**의 결합입니다.

```
[차분하고 어두운 다크 네이비 배경] 
       + [따뜻한 베이지/크림색 콘텐츠 영역] 
       + [굵은 검은색 스트로크 & 3D 입체 버튼] 
       + [장비 등급별 네온 컬러 포인트]
```

### 1.1 핵심 디자인 필러 (Design Pillars)
1. **Bold Outline (굵은 테두리)**
   - 모든 카드, 버튼, 장비 아이콘, 프로필 영역에 **2dp ~ 3dp 두께의 Dark Charcoal (#1C1A1F) 외곽선**을 적용합니다. 
   - 둥글둥글하고 만화 같은 느낌을 주어 가시성을 극대화합니다.
2. **3D Dimension Button (입체감 있는 버튼)**
   - 평평한 Flat 디자인을 배제합니다.
   - 버튼 클릭 시 눌리는 효과(Press Offset)와 버튼 하단에 **3dp ~ 4dp 깊이의 어두운 음영 단차(Shadow Border)**를 주어 물리적으로 누르고 싶게 만듭니다.
3. **Harmonious Neon Grid (네온 등급 시스템)**
   - 무채색 베이스 위에 장비의 등급(녹색, 청색, 보라색, 주황색)을 표현할 때 매우 선명하고 채도 높은 파스텔 네온 컬러를 사용합니다.
   - 특히 에픽/S급 이상의 보라색(#A259FF)과 골드/옐로우(#FFB800)의 조합을 핵심 포인트 컬러로 활용합니다.
4. **Pill-shaped Status & Currency (캡슐형 재화/스탯 바)**
   - 상단 재화 영역은 어두운 반투명 또는 차콜색 배경의 알약 모양(Pill shape) 안에 아이콘과 텍스트를 배치하고, 우측에 선명한 초록색(+) 추가 버튼을 배치합니다.

## 2. 화면별 목표 (Screen-by-Screen Visual Goals)

### 2.1 Home (로비 화면)
* **목적**: 유저의 성장 상태(특성, 랭킹 정보)와 전투 진입을 명확히 유도합니다.
* **비주얼 목표**:
  - 현재 도전 중인 챕터(예: "봉인의 전투")를 보여주는 중앙 배너는 만화책의 한 페이지처럼 강렬한 보라색/그라데이션 배경과 보스의 시그니처 몬스터(일러스트)를 중앙에 배치합니다.
  - 하단에 큼직한 **"시작"** 버튼은 굵은 테두리와 함께 그레이/옐로우 그라데이션을 적용하여 화면에서 가장 돋보이게 처리합니다.
  - 도전 기록 랭킹 영역은 1, 2, 3위 단상(포디움)을 골드, 실버, 브론즈 컬러 방패/깃발 모양 컴포넌트로 입체감 있게 구현합니다.

### 2.2 Gear / Equipment (장비 화면)
* **목적**: 유저가 수집한 아이템의 강력함을 시각적으로 뽐내고 쉽게 장착/강화하도록 돕습니다.
* **비주얼 목표**:
  - **캐릭터 쇼케이스**: 중앙에 장착된 장비와 펫/탈것을 탄 캐릭터 일러스트를 배치하고, 주변 8개 슬롯에 장착된 아이템을 배치합니다.
  - **아이템 그리드**: 장비 인벤토리는 격자형(Grid)으로 구성하며, 각 아이템 카드는 굵은 외곽선과 장비 등급 색상을 가집니다. 좌측 상단에는 무기/방어구 타입 아이콘, 우측 상단에는 강화 레벨(LV.XX), 하단에는 등급 표시(S 등)를 선명하게 박아 넣습니다.
  - 강화 가능한 장비는 우측 하단에 **초록색 위쪽 화살표(▲) 배지**를 깜빡이거나 띄워 유저의 행동을 유도합니다.

### 2.3 Upgrade (강화/특성 화면)
* **목적**: 특성 해금 및 스탯 성장감을 확실하게 전달합니다.
* **비주얼 목표**:
  - 승급 성공 시 어두운 반투명 딤(Dim) 배경 위에 캐릭터가 전면에 배치되고, 뒤쪽으로 **오렌지/골드 톤의 방사형 빛줄기(Sunburst effect)**가 퍼지는 화려한 팝업 연출을 제공합니다.
  - 이전 스탯값과 상승된 스탯값을 입체적인 베이지색 컨테이너 카드 내에 **"공격력 9340 >> 9540 (+200)"** 형태의 골드 폰트와 이펙트 기호(`>>`)로 강조하여 성장 피드백을 극대화합니다.

### 2.4 Combat (전투 화면)
* **목적**: 야추(Yahtzee) 주사위 굴리기의 손맛과 전투 진행 상황을 긴장감 있게 묘사합니다.
* **비주얼 목표**:
  - 상단 몬스터 영역은 하이브리드 캐주얼 특유의 귀여우면서도 기괴한 몬스터 일러스트와 큼직한 입체 HP 바를 제공합니다.
  - 주사위 보드는 실제 나무판 또는 가죽 매트 느낌의 텍스처 톤을 주고, 주사위는 둥근 모서리에 굵은 아웃라인과 선명한 Pip(눈)을 가진 입체 다이스로 표현합니다.
  - 주사위 선택(Keep) 시 하이라이트 테두리와 튕기는 바운스 애니메이션을 제공합니다.

### 2.5 Gacha (상점/상자 개봉 화면)
* **목적**: 상자를 열었을 때의 도파민을 극대화하고 구매 욕구를 자극합니다.
* **비주얼 목표**:
  - 상점 상품 카드는 만화책 컷처럼 굵은 빗금이나 번개 모양 백그라운드 패턴을 사용합니다.
  - **"열기 x1", "열기 x10"** 버튼은 각각 보석/골드 아이콘과 가격이 버튼 내에 입체적으로 들어가며, 10회 구매 버튼은 골드 색상으로 차별화합니다.
  - 상자가 열릴 때 상자 뚜껑이 위로 튕겨 나가며 보상이 튀어 오르는 듯한 스케일 애니메이션을 적용합니다.

### 2.6 Result (전투 결과 화면)
* **목적**: 승리의 쾌감 또는 패배의 아쉬움을 극대화하여 재도전을 유도합니다.
* **비주얼 목표**:
  - 승리 시 골드 리본 배너와 함께 트로피/장비 에셋이 화려하게 팝업됩니다.
  - 패배 시 어두운 톤의 연출과 함께 "장비 강화하러 가기" 등의 바로가기 입체 버튼을 배치합니다.

### 2.7 RunMap (챕터 진행 맵)
* **목적**: 로그라이크 진행 경로를 직관적으로 표현합니다.
* **비주얼 목표**:
  - 챕터 노드들은 굵은 점선(Path)으로 연결되어 있으며, 일반 전투, 엘리트, 보물상자, 상점, 회복 등의 아이콘이 카툰풍으로 둥글고 입체적으로 표현됩니다.
  - 현재 위치는 위아래로 둥실둥실 떠 있는 캐릭터 마커 또는 하이라이트 링 애니메이션으로 강조합니다.

## 3. 에셋 요구사항 (Asset Guidelines)

실제 이미지 파일을 직접 사용하기 어려울 경우, **Jetpack Compose의 Canvas, Vector Drawable, Shadow Modifier**를 정교하게 믹싱하여 **"코드만으로 에셋에 준하는 비주얼"**을 구현할 수 있어야 합니다.

1. **Vector Icons**:
   - 모든 검, 투구, 반지, 보석, 열쇠 등의 아이콘은 내부 채우기 컬러와 어두운 외곽선이 100% 드로잉된 스타일이어야 합니다.
   - Compose Vector가 아니더라도 `Canvas`의 `Path` 드로잉이나 기본 도형 조합에 `Stroke`를 얹어서 카툰풍 아이콘을 직접 생성할 수 있어야 합니다.
2. **Dynamic Canvas Drawing**:
   - 주사위 입체 눈, 주사위 컵, 캐릭터 실루엣(Paperdoll 바디) 등은 Canvas를 활용해 동적으로 그리고, 유연하게 회전/크기 변환 애니메이션이 가능하도록 설계합니다.

## 4. 따라 하면 안 되는 저작권/카피 가드레일 (Copyright Guardrails)

* **주의**: 궁수의 전설(Archero)이나 탕탕특공대(Survivor.io)의 비주얼 스타일(Style)은 차용하되, **그들의 고유 캐릭터 디자인, 특정 에셋 리소스 복제, 명칭(예: 'Survivor' 등)을 그대로 사용해서는 안 됩니다.**
* **안전한 디자인 적용법**:
  - **에셋 카피 금지**: 리퍼런스 이미지의 캐릭터나 장비를 픽셀 단위로 베끼지 말고, Yacoo RPG만의 오리지널 히어로(초록색 곰, 외눈박이 공허의 눈 등)와 독창적인 카툰풍 아이콘을 새로 그립니다.
  - **레이아웃 변형**: 화면 레이아웃(상단 재화 바 위치, 하단 5탭 구성 등)은 모바일 RPG의 표준 UX(De-facto Standard)이므로 뼈대는 참고하되, 디테일한 장식적 요소나 색상 배치 비율을 다르게 설계하여 독창성을 확보합니다.

## 5. 비주얼 성공 기준 (Success Criteria)

1. **Aesthetic Wow Factor**: 화면을 처음 켰을 때 기본 Android 머티리얼 UI가 연상되지 않고, **"상용 인디 모바일 게임 빌드"**로 보일 만큼 완성도 있는 테마가 일관되게 적용되었는가?
2. **Tactile Feedback (손맛)**: 모든 버튼 클릭 시 실제 1~2dp 아래로 쑥 내려가는 애니메이션과 미세한 진동/효과음(구현 가능한 경우) 연출이 적용되어 있는가?
3. **Outline Consistency**: 팝업창, 툴팁, 탭바 등 화면 내에 아웃라인이 생략되거나 어색하게 평평한(Flat) 컴포넌트가 방치되어 있지 않고 일관된 두께의 Dark Charcoal Outline 규칙을 지키고 있는가?
