# Arrow Puzzle (Tap Away) - Game Specification

## 1. 개요 (Overview)
- **장르:** 퍼즐 (Unravel / Unblock 스타일)
- **목표:** 얽혀 있는 화살표(선)들을 클릭하여 화면 밖으로 모두 빼내는 게임.
- **플랫폼:** 웹 브라우저 (PC/Mobile 호환)
- **기술 스택:** HTML5 `<canvas>`, Vanilla JavaScript (ES6 modules), CSS3, Vite

## 2. 핵심 규칙 (Core Mechanics)
1. **블록(화살표 선)의 구성:**
   - 각 블록은 꼬불꼬불하거나 꺾인 '경로(Path)'를 가지고 있으며, 끝에는 방향을 나타내는 '화살표(Arrow head)'가 있다.
2. **이동 (Movement):**
   - 플레이어가 화살표의 머리나 몸통을 클릭/터치하면, 해당 선은 자신이 향하는 궤적을 따라 앞으로 전진하며 화면 밖으로 빠져나간다.
3. **충돌 및 막힘 (Collision & Blocking):**
   - 빠져나가려는 선의 경로 앞을 **다른 선이 가로막고 있다면** 이동할 수 없다. (클릭 시 막혔다는 피드백 - 예: 흔들림 애니메이션)
   - 반드시 앞에 장애물이 없는 선부터 순차적으로 빼내야 한다.

## 3. 구현 단계 (Phases)
- **Phase 1: 프로토타입 (직선 화살표)**
  - 직선으로만 이루어진 화살표 3~4개를 겹치게 배치.
  - 클릭 이벤트 감지 및 직선 이동 애니메이션.
  - 선분 기반 단순 충돌(교차) 판정.
  - 현재 구현 상태: 완료. 기본 화살표 생성, 클릭/터치, 이동, 충돌 판정이 동작한다.
- **Phase 2: 곡선/꺾임 이동 로직**
  - 화살표가 꺾인 경로를 따라 이동하는 Path 애니메이션 로직 구현.
  - 복합 다각형/선분 그룹 간의 정밀한 충돌 처리.
  - 현재 구현 상태: 꺾인 직선 세그먼트 체인은 구현됨. Bezier 곡선 이동이나 연속 충돌 감지는 별도 구현되지 않았다.
- **Phase 3: 레벨 디자인 & 비주얼 요소**
  - 코끼리, 모나리자 등 복잡한 형태를 구성하기 위한 레벨 데이터 구조화 (JSON 방식 등).
  - 색상, 부드러운 이동 애니메이션, 클릭 이펙트(파티클 등) 추가.
  - 현재 구현 상태: 이미지 기반 마스크 생성, JSON Export, 파티클, Web Audio, Undo가 구현됨. 수동 레벨 로드/스테이지 진행은 아직 없다.

## 4. 파일 구조 (File Structure)
- `index.html`: 게임 화면 렌더링용 Canvas 컨테이너
- `style.css`: 게임 화면 레이아웃 및 다크 테마 배경
- `main.js`: 캔버스 제어, 게임 루프(update/draw), 이벤트 리스너, 이미지 마스크 처리, 레벨 생성, Undo/Export 관리
- `Arrow.js`: 화살표 경로, 렌더링, 이동, 히트 테스트, 충돌/탈출 판정
- `tests/arrow.test.js`: Arrow 기하/충돌 단위 테스트
- `package.json`, `vite.config.js`: Vite/Vitest 실행 및 빌드 설정
